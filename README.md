# Formulário de Levantamento de Requisitos

Este projeto tem como finalidade auxiliar no **levantamento de requisitos** para o desenvolvimento do TCM do curso técnico.  
O formulário foi pensado para ser enviado ao coordenador, que neste contexto representa o cliente (dono de uma agência de viagens), permitindo coletar de forma organizada as informações necessárias para entender as necessidades do sistema.

A aplicação foi desenvolvida utilizando **HTML, CSS e JavaScript**, com foco em uma estrutura simples e compatível com o nível técnico atual do projeto.

Para agilizar o desenvolvimento, foi utilizada **Inteligência Artificial na criação do formulário**, e posteriormente o conteúdo foi **adaptado, revisado e alterado conforme as necessidades específicas do projeto**.

## Objetivo
- Facilitar a coleta de requisitos
- Organizar as respostas do cliente
- Apoiar a etapa de análise do sistema
- Servir como base para as próximas fases do TCM

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript
- Formspree
